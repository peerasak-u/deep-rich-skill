# journal

Log buy/sell reasoning; review outcomes; find patterns.

## When

- After any buy/sell
- "why did I buy X" / "what was my thesis"
- Monthly outcome review

## File

`.deep-rich/journal.md`

```markdown
## YYYY-MM-DD — BUY/SELL/HOLD <SYM>
**Price:** $X | **Amount:** ฿X (N shares)
**Thesis:** … | **Risk:** … | **Target:** …
**Health:** A–F | **Allocation:** % of class
### Outcome (later)
**Price:** $X | **Gain/Loss:** ±X% | **Thesis valid?:** Y/N/Partial | **Lesson:** …
```

## Log flow

1. `dr fundamentals <SYM>` + `dr health <SYM>`
2. Ask: why buy/sell? target? sell trigger? risk?
3. Append entry

## Review flow

1. Read `journal.md`
2. `dr performance` — gain since decision, thesis valid?, risk materialized?
3. Patterns: win rate, avg winner/loser, biggest mistake, what works/fails
4. Synth: recent decisions → patterns → strengths → improvements

## Guards

Log every trade; honest reasoning; monthly review; learn from mistakes; no judgment.

Tools: `dr fundamentals`, `dr health`, `dr performance`, `web_search`, `test_conviction_with_sentiment()`
